# Python Foundations

A reference repository containing core Python scripts, syntax exercises, and object-oriented programming concepts. This serves as a working notebook for foundational mechanics before applying them to larger software projects.

## Repository Structure

* **`main.py`** Core syntax, loops, conditional logic, basic data structures (lists, dictionaries, sets), and file I/O operations.
* **`advance.py`** Functional programming elements including list/dictionary comprehensions, decorators, dunder methods, lambda functions, map, and filter.
* **`oops.py`** Object-Oriented Programming (OOP) architecture, including class initialization, inheritance, method overriding, and abstraction.

## Environment
* Python 3.x
